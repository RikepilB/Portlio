# PRODUCT.md — Richard Pillaca Portfolio

register: brand

## Product Purpose

Personal portfolio for Richard Pillaca Burga (Software & Data Engineer, Toronto). Presents
data and engineering case studies with the tactile restraint of premium packaging — sage felt,
gold foil, balanced whitespace.

## Users

- Recruiters and hiring managers scanning for evidence-led project depth
- Technical peers evaluating stack choices and systems thinking
- Collaborators discovering personal context (About vision board)

## Brand principles

- Material, not neon: matte felt + metallic foil, never terminal-cyber aesthetics
- Evidence-led: `src/data/` owns all metrics and project claims
- Structure preserved: v2 page composition is immutable; v3 is a re-skin
- Accessible polish: motion is supplementary; reduced-motion always works

## Anti-references

- Generic SaaS dashboards
- Neon terminal / cyberpunk developer themes
- Velvet-charcoal/copper editorial (rejected v3 attempt)
- Invented metrics or placeholder Lorem Ipsum

## Tone

Precise, composed, warm-technical. Short declarative sentences. No hype adjectives.
