# Portfolio Identity System Tokens

## Typography Context Mapping
*   Display Serif Font (The High-Contrast Editorial Serif): Used strictly for major project titles like "ScoutLane," "Empeñalo," or analytical hooks.
*   Clean UI Sans-Serif (The Geometric Wide-Spaced Font): Used for technical descriptions, code snippets, system architecture logs, and navigation items.

## Color & Texture Mapping
*   Base Velvet Canvas: A deep, muted dark charcoal or rich stone grey base that gives the site a premium, developer-centric studio environment.
*   Foil Highlights: Reflective gold or copper-gold foil paths applied exclusively to key interactive nodes, terminal headers, or critical layout highlights.
