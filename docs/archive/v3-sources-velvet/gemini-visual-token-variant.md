# Visual Identity Tokens — Richard Pillaca

## 1. Color Palette System (Tailwind v4 Setup)
*   **Background Base (Muted Velvet Charcoal):** `#1C1A19` (A soft, deep stone gray designed to absorb light on the display)
*   **Card Container Default (Muted Espresso):** `#262221` (For default project blocks, creating structural depth over the base)
*   **Card Highlight Base (Dewy Soft Slate):** `#E1E8E9` (High-contrast bright element surfaces used selectively)
*   **Metallic Accent (Copper Gold Foil):** Base `#C58E7F`, Highlight `#FAECE9`, Shadow `#735147` (For interactive typography paths and dynamic lighting gradients)
*   **Core Text (Matte Chalk White):** `#F4F4F3`
*   **Subdued Labels (Muted Earth Grains):** `#8C8380`

## 2. Typography System
*   **Display Typography (Editorial Serif):** *Ogg* or *Playfair Display*. High stem contrast, razor-thin serifs, tight tracking (`tracking-tight`), and prominent bold weight. Reserved for page titles and main project titles.
*   **Labels & Indicators (Wide Classical Serif):** *Cormorant Garamond* or *Cinzel*. All-caps text styling with wide character tracking (`tracking-[0.25em]`). Reserved for technical categories, tags, and runtime metrics.
*   **System UI & Body text:** *Manrope* or *Inter*. Minimalist, highly legible geometric sans-serif font family used for technical descriptions, code strings, logs, and layout headers.
