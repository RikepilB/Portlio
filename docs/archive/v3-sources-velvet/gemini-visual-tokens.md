# Visual Identity Tokens — Richard Pillaca

## 1. Color System
*   Background Canvas (Muted Velvet Charcoal): `#1C1A19` (Soft, deep matte texture base)
*   Card Surface Light (Dewy Soft Slate): `#E1E8E9` (For high-contrast elements like the About block)
*   Card Surface Dark (Muted Espresso Stone): `#262221` (Default container fill)
*   Reflective Accent (Copper Gold Foil): Base `#C58E7F`, Highlight `#FAECE9`, Shadow `#735147`
*   Text Core Light (Matte Chalk): `#F4F4F3`
*   Text Subdued (Muted Earth Grains): `#8C8380`

## 2. Font System (Next.js config mapping)
*   Display Serif (The High-Contrast Editorial Headings): *Ogg* or *Playfair Display* styled with tight tracking (`tracking-tight`) and bold weights. Used for page titles and project names.
*   Labels & Accents (Wide Classical Serif): *Cormorant Garamond* or *Cinzel* (All-caps, `tracking-[0.25em]`). Used for small technical labels, tags, and category names.
*   Body Text (Clean Geometric Mono/Sans): *Manrope* or *Inter* (Clean, ultra-legible, modern sans). Used for descriptions, code logs, metrics, and labels.
