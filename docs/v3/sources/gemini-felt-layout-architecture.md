# Structural Layout Architecture — richardpillaca.com

## 1. Spatial Layout Rules
*   **Center-Balanced Alignment:** The text hierarchy across your home landing view is strictly aligned dead-center. It implements generous padding limits (`py-24`, `px-12`) to keep elements isolated by spacious blocks of clean whitespace.
*   **Soft Side Lighting:** Web blocks utilize an off-center radial ambient shadow casting gently from the top-left toward the bottom-right. This creates a realistic, three-dimensional look across the screen.

## 2. Tactile Textures
*   **The Suede Felt Simulation:** Solid background panels are layered under an SVG linen/canvas texture set to a low opacity (3% to 5%). This technique mimics the fuzzy, light-absorbing qualities of tactile luxury cardboard packaging.
