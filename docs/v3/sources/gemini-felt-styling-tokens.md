# Comprehensive Styling System Tokens — Richard Pillaca

## 1. Primary Canvas Theme (Home & Architecture)
*   **Felt Casing Background (Sage Green-Grey Mix):** `#607466` or `#708275` (Muted, sophisticated olive-tinted felt tone)
*   **Primary Foil Element (Metallic Yellow Gold):** Base `#D4AF37`. Interactive reflection uses a dynamic CSS gradient: `linear-gradient(135deg, #F2E3C6, #D4AF37, #AA7C11)`
*   **Core Typography Light:** `#FFFFFF` (Stark, flat matte white)

## 2. Complementary Theme (About Vision Board)
*   **Satin Backdrop (Pearl Grey / Off-White):** `#FDFBF7` or `#EAECEB` (Cool, ultra-clean presentation layout)
*   **Secondary Foil Element (Reflective Rose Gold):** Base `#C58E7F`, Highlight `#FAECE9`, Shadow `#735147`

## 3. Font System Strategy
*   **Main Display Title (The Clean Bold Sans):** *Manrope* or *Montserrat* styled with tall capital letters, heavy weighting, and wide tracking (`tracking-[0.18em]`). Used for your main name banner and core project headings.
*   **Regal Script & Secondary Accents (Classical Logo Serif):** *Cormorant Garamond* (Italic variant) or *Cinzel* for small uppercase text tracking. Used for branding text blocks and metadata accents.
